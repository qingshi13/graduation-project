#### 使用说明
此项目是前后端分离项目，前后端需要分别启动。

##### 前端启动  
首次启动，进行入目录 client-ui 下执行命令 npm install 
之后启动仅需执行命令 npm run dev

##### 后端启动  
普通的spring boot项目配置maven加载好依赖即可启动
